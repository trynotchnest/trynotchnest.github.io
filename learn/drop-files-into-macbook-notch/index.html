How to Drop Files Into the MacBook Notch

The notch can catch files you drag onto it — a tray for AirDrop and quick sharing. Here's how MacBook notch file drop works and why it's faster than Finder.

Updated June 15, 20265 min readBy Satnam Singh

Dragging a file to share it usually means juggling Finder windows or the share sheet. MacBook notch file drop replaces that: you drag a file onto the notch and it becomes a launch point for AirDrop, a holding tray, or a quick action.

How notch file drop works

With NotchNest running, the notch becomes a drop target. Pick up any file — from Finder, the Desktop, or an app — drag it to the notch, and a tray opens:

  1. AirDrop — release on a nearby device to send instantly.
  2. File tray — park files there and grab them again later from any app.
  3. Quick share — hand off to the share sheet without hunting for the menu.

Why it beats the normal flow

The notch is always at the top center of your screen, so it's a fixed, predictable target no matter which app is in front. No window switching, no right-click menus. Drag up, drop, done. The recent-files tray means you don't lose track of what you moved.

Frequently asked

Can you drag files onto the MacBook notch?
Yes, with a notch app like NotchNest. Dragging a file to the notch opens a tray for AirDrop, a holding shelf, or the share sheet.
Is notch file drop faster than AirDrop from Finder?
Usually. The notch is a fixed target at the top center of every screen, so there's no window switching or right-click menu — drag up and release.
Does the notch tray keep my files?
The file tray holds recents so you can grab them again from any app. Nothing is uploaded; files stay local on your Mac.
Which Macs support notch file drop?
Any notched MacBook — MacBook Pro 2021+ and MacBook Air 2022+ — running macOS 14 or later with NotchNest installed.
